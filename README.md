```pip install gh-download-cli```
<br>
to start <br>
```ghdownload```


- Website: https://rhithesh.github.io/ghdownload/

## Resetting Your Token

If you want to reset your saved GitHub token, run:

```bash
rm ~/.ghdownloadertoken
```
